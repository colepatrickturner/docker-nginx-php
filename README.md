# NGINX | PHP7 - for Docker:

# Features

#### NGINX

Content is served from `/app/src` and the default VHOST is configured with HTTP2 support.

#### PHP-FPM

Default configuration for FPM and integration with default VHOST.


#### Addons
##### Included
- V8JS

##### Planned
- Firewall
- Fail2ban

# Environment

**MEMCACHED_MEMORY_LIMIT** [128]
